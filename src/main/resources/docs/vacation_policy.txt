# 2024년도 유급 휴가 및 휴직 규정

시행일: 2024년 1월 1일
적용 대상: 전 임직원 (정규직, 계약직 포함)
관리 부서: 인사팀

## 1. 연차 유급휴가

### 1-1. 기본 부여 기준
- 입사 1년 미만: 1개월 개근 시 1일 발생 (최대 11일)
- 입사 1년 이상: 연간 15일 일괄 부여
- 근속 3년 이상: 매 2년마다 1일 추가 가산 (최대 25일 한도)
  예) 3년차: 16일 / 5년차: 17일 / 10년차: 19일 / 20년차: 25일

### 1-2. 발생 및 소멸 기준
- 발생 기준일: 입사일 기준
- 소멸 기준: 미사용 연차는 해당 연도 말일(12월 31일) 자정 자동 소멸
- 이월 불가: 잔여 연차는 다음 연도로 이월되지 않음
- 단, 회사의 귀책사유로 사용하지 못한 연차는 연차수당으로 보상

### 1-3. 연차 사용 촉진 제도
- 회사는 연차 소멸 6개월 전 잔여 일수를 개인에게 서면 통보
- 통보 후 10일 이내 직원이 사용 일정을 제출하지 않을 경우, 회사가 사용 시기를 지정할 수 있음
- 사용 촉진 절차를 이행한 경우, 미사용 연차에 대한 수당 지급 의무 면제 가능 (근로기준법 제61조)

### 1-4. 연차 일할 계산
- 중도 퇴사자의 경우, 재직 기간에 비례하여 연차를 정산하며 초과 사용분은 급여에서 공제

## 2. 병가 및 특별 휴가

### 2-1. 유급 병가
- 부여 일수: 연간 최대 10일 (역년 기준 1월 1일 ~ 12월 31일)
- 급여 지급: 병가 기간 중 통상임금의 100% 지급
- 진단서 제출 의무:
  · 1~2일: 제출 불필요 (구두 보고로 갈음 가능)
  · 3일 이상 연속 사용: 의료기관 발급 진단서 및 소견서 제출 필수
  · 5일 이상: 입원확인서 또는 입원 영수증 추가 제출

### 2-2. 무급 병가
- 유급 병가 10일 초과 시, 무급 병가로 전환 가능 (최대 30일)
- 장기 치료가 필요한 경우 별도 휴직 신청 가능 (아래 휴직 규정 참고)

### 2-3. 병가 신청 절차
- 원칙: 사전 ERP 시스템 등록 및 팀장 승인
- 긴급 상황 시: 당일 유선 또는 메신저로 팀장에게 보고 후, 복귀 당일 사후 ERP 등록
- 허위 신청 또는 허위 서류 제출 시 징계 처분 대상

### 2-4. 공무상 병가
- 업무 중 발생한 부상 또는 질병의 경우, 산업재해 처리 후 산재 기간 동안 별도 병가 일수 차감 없음

### 2-5. 특별 휴가 (무급)
- 개인 사정으로 인한 단기 휴가 필요 시, 팀장 협의 후 무급 특별 휴가 신청 가능
- 연간 최대 5일 한도 (단, 업무 공백이 최소화되는 시기에 한함)

## 3. 경조사 휴가

### 3-1. 경조사 휴가 일수 상세

| 구분   | 대상                          | 휴가 일수              |
|--------|-------------------------------|------------------------|
| 결혼   | 본인                          | 5영업일                |
| 결혼   | 자녀                          | 1영업일                |
| 결혼   | 형제 / 자매                   | 1영업일                |
| 사망   | 배우자 / 부모 / 자녀          | 5영업일                |
| 사망   | 조부모 / 외조부모             | 3영업일                |
| 사망   | 형제 / 자매                   | 1영업일                |
| 출산   | 배우자 출산 (남성 직원)       | 10영업일               |
| 출산   | 본인 출산 (여성 직원)         | 출산전후휴가 별도 적용 |
| 입양   | 본인 입양                     | 5영업일                |

### 3-2. 경조사 휴가 사용 규정
- 경조사 발생일로부터 30일 이내 사용을 원칙으로 하며, 기간 초과 시 소멸
- 경조사 휴가는 연차에서 차감되지 않으며, 별도 부여
- 사용 전 증빙 서류 제출 필수 (청첩장, 사망진단서, 출생증명서 등)
- 휴일·공휴일과 겹치는 경우 영업일 기준으로 산정

### 3-3. 배우자 출산휴가 세부 규정
- 10영업일은 분할 사용 가능 (최대 3회 분할)
- 출산일로부터 90일 이내에 전부 사용해야 하며, 기간 초과 시 잔여 일수 소멸
- 급여는 통상임금의 100% 지급 (정부 지원금 포함)

## 4. 법정 의무 휴가 및 휴직

### 4-1. 출산전후휴가 (여성 직원)
- 기간: 출산 전후 90일 (다태아 120일), 출산 후 최소 45일 이상 사용
- 급여: 최초 60일은 통상임금 100% 지급 (회사 부담), 이후 30일은 고용보험 적용

### 4-2. 육아휴직
- 대상: 만 8세 이하 또는 초등학교 2학년 이하 자녀를 둔 임직원
- 기간: 최대 1년 (부부 각각 1년씩 사용 가능)
- 급여: 고용보험 육아휴직급여 기준 적용
- 복직 보장: 육아휴직 종료 후 동일 또는 동등한 직위·직무로 복직 보장

### 4-3. 가족돌봄휴가
- 부모, 배우자, 자녀, 조부모 등 가족의 질병·사고·노령 시 사용 가능
- 연간 최대 10일 (무급), 단 1일 단위 사용 가능

### 4-4. 배우자 사망 후 심리 지원 휴가
- 경조사 5일 외 추가로 3일의 심리 안정 휴가 부여 (유급)

## 5. 휴가 신청 절차 및 유의사항

### 5-1. 일반 신청 절차
1. 사내 ERP 시스템(HR 포털) 접속
2. [휴가 신청] 메뉴에서 유형 선택 및 날짜 입력
3. 사유 기재 후 팀장 결재 요청
4. 팀장 승인 완료 후 일정 확정
   (미승인 상태에서 결근 시 무단결근 처리 가능)

### 5-2. 신청 기한
- 연차 휴가: 최소 3영업일 전 신청 권장 (긴급 시 당일 신청 가능, 팀장 재량)
- 경조사 휴가: 사전 또는 사후 3영업일 이내 증빙 제출
- 육아휴직: 시작 예정일 최소 30일 전 서면 신청

### 5-3. 승인 거부 및 시기 변경
- 팀장은 업무상 긴박한 사유가 있을 경우 사용 시기 변경을 요청할 수 있음
- 단, 거부는 불가하며 반드시 대체 사용 일정을 협의하여 제시해야 함
- 시기 변경 요청은 서면(ERP 메모 또는 이메일)으로 기록 남길 것

### 5-4. 유의사항
- 무단결근(사전 신청 및 사후 보고 없는 결근)은 인사 불이익 처리
- 허위 사유 기재 또는 서류 위조 시 징계위원회 회부
- 미사용 연차수당은 다음 연도 2월 급여일에 일괄 정산 지급
- 반차(오전/오후 0.5일) 사용 가능하며, 반차 2회 사용 시 1일 차감
